# MIRAE Brandbook

*Design & Development Reference for Web App*

---

## 1. Brand Essence

### Brand Meaning
**Mirae (미래)** means *“future”* in Korean.

The brand represents:
- Calm exploration, not pressure
- Soft guidance, not authority
- Emotional safety, not evaluation
- Curiosity, optimism, and gentle momentum

### Personality
| Trait | Expression |
|---|---|
| Gentle | Rounded shapes, soft gradients |
| Trustworthy | Clean layouts, generous spacing |
| Optimistic | Pastel spectrum, warm highlights |
| Non-judgmental | No harsh contrasts, no sharp edges |
| Friendly intelligence | Playful icon, restrained UI |

---

## 2. Visual Language Overview

### Core Principles
1. Soft over sharp
2. Gradient over flat
3. Calm over loud
4. Whitespace as breathing room
5. Motion should feel like floating, not snapping

---

## 3. Color System (CSS Reference)

### Core Gradient Palette
```css
--mirae-blue: #9BCBFF;
--mirae-lavender: #C7B9FF;
--mirae-pink: #F4A9C8;
--mirae-peach: #FFD1A8;
--mirae-mint: #BEEDE3;
```

### Primary Brand Gradient
```css
background: linear-gradient(
  135deg,
  #9BCBFF 0%,
  #C7B9FF 25%,
  #F4A9C8 50%,
  #FFD1A8 75%,
  #BEEDE3 100%
);
```

## 5. Components & Motion

Everything floats. Nothing snaps.

```css
transition: all 0.3s ease-out;
```

---

## North Star Rule

If it feels calm, friendly, and slightly magical — it’s correct.
If it feels efficient but cold — it’s wrong.
